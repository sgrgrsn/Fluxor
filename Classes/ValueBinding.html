ValueBinding

public class ValueBinding<Value, UpdateValue>

An ValueBinding can be used as a two-way connection between a slice of the Store and a View. The current property will always hold the current Value in the State and when one of the update functions are called, an Action is dispatched on the Store.

    The current value. This will change everytime the State in the Store changes
